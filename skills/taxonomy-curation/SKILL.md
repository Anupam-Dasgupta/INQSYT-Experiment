# Purpose

Improve the extracted taxonomy by proposing merges, splits, renames, additions, or deletions.

---

# Inputs

- PAGE_INTENT_CANDIDATES

---

# Outputs

- PAGE_INTENT_CHANGE_PROPOSALS
- Updated taxonomy.md
- Updated curation.md

---

# Responsibilities

- Identify duplicate intents.
- Propose merges.
- Propose splits.
- Propose renames.
- Justify every proposal.
- Record every proposal inside curation.md.

---

# Must Not

- Apply changes without approval.
- Remove evidence.
- Invent unsupported intents.
